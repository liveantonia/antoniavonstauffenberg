# Images

Place the following image files in this directory:

| File | Description |
|------|-------------|
| `hero-bg.jpg` | Full-width hero/scroll header background (recommended: deep-sky / galaxy image, ≥ 2000 px wide) |
| `research-bg.jpg` | Research banner parallax background (recommended: astronomical image, ≥ 1600 px wide) |
| `portrait.jpg` | Your portrait photo (recommended: 3:4 portrait crop) |
| `cv.pdf` | *(optional)* If placed here, the CV page will load it automatically |

> **Tip:** Free high-quality astronomy images are available from [NASA Image Gallery](https://images.nasa.gov/), [ESA/Hubble](https://esahubble.org/images/), and [ESO](https://www.eso.org/public/images/).
